The Process

With all that is shown in this website, you might be asking yourself; How did this designer make all this stuff from scratch? The truth is it aint easy initially. The reason being is that initially, you have so many ideas that you don't know where to start, and there is nothing to help you begin the process. This makes it hard for programmers of all levels to design the website at the start, but over time everything starts working out and you get a better picture as to what your website actually is. This website was designed using HTML and CSS as the languages, and written on the Sublime Text Software in order to program it. I made several html files to create pages for different sections, these can be found in the navbar. All the pages use a single css file which allow for the websites to look better than with just standard html as it allows the features of the parts of my website to be customised to whatever I like.
